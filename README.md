# 💖 For Rahaf - Stay With Me

A premium, high-end web experience dedicated to Rahaf, featuring elegant typography, smooth animations, and a sophisticated aesthetic.

## ✨ Features
- **Premium Typography**: Uses 'Cinzel Decorative' for a classic, high-end feel.
- **Dynamic Animations**: Smooth transitions, shaking effects, and a dramatic "break" sequence.
- **Glassmorphism & Glow**: Modern design with subtle glow effects and premium framing.
- **GitHub Pages Ready**: Optimized for instant deployment on GitHub.

## 🚀 How to Publish on GitHub Pages
1. Create a new repository on GitHub named `rahaf-gift`.
2. Upload the `index.html` file to that repository.
3. Go to **Settings** > **Pages**.
4. Under **Build and deployment**, set **Source** to "Deploy from a branch".
5. Select the `main` branch and `/ (root)` folder, then click **Save**.
6. Your site will be live at `https://<your-username>.github.io/rahaf-gift/`.
